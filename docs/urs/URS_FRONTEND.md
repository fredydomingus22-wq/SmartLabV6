

## 1. Plataforma

O frontend deve:

- Ser construído em Next.js 15
- Usar App Router
- Ser responsivo
- Ser responsivo (Prioridade: Tablet Industrial-First 768px-1024px)

## 2. Estrutura

Módulos visuais:

- Dashboard
- Produção
- Laboratório
- Especificações
- Parâmetros
- Reagentes
- Auditorias
- Relatórios

## 3. Funcionalidades Gerais

Cada módulo deve permitir:

- Criar
- Editar
- Versionar
- Tornar obsoleto
- Ver histórico de alterações
